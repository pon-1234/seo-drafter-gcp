# 記事テンプレート集

初心者向けSEO記事を簡単に作成するためのテンプレート集です。

## 📚 テンプレート一覧

### 1. 初心者向け「◯◯とは」記事
**ファイル**: `beginner-information.json`

**用途**:
- 「デジタルマーケティングとは」などの入門クエリ
- 基本概念の説明記事
- 定義→手法→メリデメ→始め方の王道構成

**推奨キーワード例**:
- デジタルマーケティングとは
- SEOとは
- コンテンツマーケティングとは

**構成**:
1. 30秒で要点
2. ◯◯の意味をわかりやすく解説
3. ◯◯の主な手法と役割（表で比較）
4. ◯◯のメリット・デメリット
5. ◯◯を始める5ステップ
6. よくある失敗と対処法
7. FAQ
8. まとめ

---

### 2. 初心者向けハウツー記事
**ファイル**: `beginner-howto.json`

**用途**:
- 「Googleアナリティクス 設定方法」などの実践クエリ
- 手順解説記事
- ステップバイステップの説明

**推奨キーワード例**:
- Googleアナリティクス 設定方法
- SEO対策 やり方
- WordPress ブログ 作り方

**構成**:
1. 30秒で要点（完成イメージ）
2. 始める前に準備するもの
3. ◯◯の手順を5ステップで解説
4. 各ステップの詳細
5. よくあるつまずきポイントと解決法
6. FAQ
7. まとめ（次のステップ）

---

### 3. 初心者向け比較・おすすめ記事
**ファイル**: `beginner-comparison.json`

**用途**:
- 「マーケティングツール おすすめ」などの比較クエリ
- 選び方ガイド
- ランキング・レビュー記事

**推奨キーワード例**:
- マーケティングツール おすすめ
- SEOツール 比較
- メール配信サービス おすすめ

**構成**:
1. 30秒で要点（結論：おすすめTOP3）
2. ◯◯を選ぶポイント3つ
3. おすすめTOP5を比較（表）
4. 各ツールの詳細レビュー
5. 使う人別のおすすめ
6. よくある質問（FAQ）
7. まとめ（選び方のチェックリスト）

---

## 🎯 共通の特徴

すべてのテンプレートは以下の特徴を持っています：

- **expertise_level**: `beginner`（初心者向け）
- **tone**: `friendly`（親しみやすい）
- **文字数**: 各セクション400〜600字
- **1文の長さ**: 60字以内を厳守
- **専門用語**: 必ず言い換え＋具体例
- **出典**: 各セクション末尾に1〜2本のみ
- **参照ソース**: 総務省、経産省、Google公式、Wikipedia、ferret、バズ部など初心者向け

## 📝 使い方

### 方法1: UIからテンプレートを選択（予定）
1. Brief入力フォームで「テンプレートを使う」をクリック
2. 目的に合ったテンプレートを選択
3. primary_keywordだけ入力すればOK

### 方法2: APIで直接指定
```bash
curl -X POST "https://seo-drafter-api-468719745959.asia-northeast1.run.app/api/jobs" \
  -H "Content-Type: application/json" \
  -d @templates/beginner-information.json
```

### 方法3: テンプレートの内容をコピー
JSONファイルの内容をBrief入力フォームにコピーして使用

## 🔧 カスタマイズ

テンプレートは必要に応じてカスタマイズできます：

1. **primary_keyword**: 実際のキーワードに置き換え
2. **persona_brief.needs**: 読者のニーズを追加・変更
3. **notation_guidelines**: プロジェクト固有のルールを追加
4. **recommended_structure**: セクション構成を調整

## 📊 テンプレート選びのヒント

| 検索意図 | テンプレート | 例 |
|---------|------------|-----|
| 知りたい（What） | beginner-information | ◯◯とは、◯◯の意味 |
| やりたい（How） | beginner-howto | ◯◯ やり方、◯◯ 設定方法 |
| 選びたい（Which） | beginner-comparison | ◯◯ おすすめ、◯◯ 比較 |
